## How to deploy the files:

```shell
ln -f -s "$HOME/Documents/GitHub/dotfiles/gemini/GEMINI.md" "$HOME/.gemini"
ln -f -s "$HOME/Documents/GitHub/dotfiles/gemini/antigravity-cli/settings.json" "$HOME/.gemini/antigravity-cli"
ln -f -s "$HOME/Documents/GitHub/dotfiles/gemini/antigravity-ide/settings.json" "$HOME/Library/Application Support/Antigravity IDE/User"
```
